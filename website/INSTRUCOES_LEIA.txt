Nesse folder está a descrição de como essa pagina irá funcionar

1-Botão Adicionar chachorro: esse botão leva o usuário para a pagina de cadstro de cachorros,onde pode colocar um novo animal para adoção.

2-Botão Remover cachorro: permite remover um animal do catalogo de adoção que será mostrado para quem deseja adotar.

3-Botão Editar dados:Leva o usuário para a pagina de cadastro para poder fazer qualquer alteração na descrição do cachorro selecionado.

4-Botão Minha Conta: redireciona o usuário para a página de sua conta onde poderá fazer qualquer ajuste relacionado aos seus dados pessoais.

5-Botão Meus cachorros: Mostra essa página que voce está vendo no momento

6-Botão Carregar cachorros: carrega todos os cards contendos cachorros colocados para adoção
